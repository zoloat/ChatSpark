MCV モード

MultiCommentViewer 経由でコメントを受信

OBS ブラウザソース URL: http://localhost:39080/display  |  Effect Craft を開く  |   |  モード選択に戻る

1. MCV 接続ステータス

● 未接続

MCVCommentRelay プラグインが接続されていません。
MCV を起動し、プラグインの接続先が http://localhost:39080 になっていることを確認してください。

● 接続中

MCVCommentRelay プラグインからコメントを受信できます。

2. トリガー設定

MCV 経由ではチャットキーワードのみ使用できます。チャンネルポイントは Twitch モードで設定してください。

3. プレビュー

4. OBS への追加方法

  1. OBS の「ソース」パネルで「＋」をクリック →「ブラウザ」を選択
  2. URL に http://localhost:39080/display を入力
  3. 幅と高さを配信画面と同じ解像度にする（例: 1920 × 1080）
  4. 背景は自動で透過されるので、そのまま他のソースに重ねて OK

画像一覧は設定保存時に記録されます。画像を追加・削除した場合は保存を押し直してください。
