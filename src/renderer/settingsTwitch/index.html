Twitch モード

Twitch チャット・チャンネルポイントに直接接続

OBS ブラウザソース URL: http://localhost:39080/display  |  🎨 Effect Craft を開く  |   |  モード選択に戻る

📖 使い方 クリックで開く

▶ 初回セットアップ手順

  1. Twitch Developer Console（https://dev.twitch.tv/console/apps）にアクセス
  2. 「Register Your Application」でアプリを新規作成
      ・名前: 任意（例: my-overlay）
      ・OAuth リダイレクト URI: http://localhost:39080/oauth/callback
      ・カテゴリ: Chat Bot
      ・機密保持について: パブリック（Client Secret 不要のため）
  3. 作成後に表示される クライアント ID をコピー
  4. このページの「0. Twitch アプリ設定」に貼り付けて「保存」
  5. 「1. Twitch 認証」で「Twitch に接続」を押してログイン
  6. Effect Craft（effect-craft/index.html）でエフェクトを作成して JSON をエクスポート
  7. 「2. トリガー設定」でキーワードとエフェクト JSON を設定して「設定を保存」
  8. OBS のブラウザソースに http://localhost:39080/display を登録（詳しくは下の「4. OBS への追加方法」）

▶ トリガータイプ

タイプ 説明
チャット キーワード チャットに指定したワードが含まれると発動
チャンネルポイント チャンネルポイント報酬の名前と一致すると発動

▶ エフェクト設定

Effect Craft（effect-craft/index.html）でエフェクトを作成し、JSONをエクスポートして各トリガーに読み込んでください。

▶ 画像の使い方

  1. 使いたい画像ファイル（PNG・GIF など）を以下のフォルダに置く
    %APPDATA%\twitch-overlay-support\assets\
  2. Effect Craft の「Image」欄にそのファイル名を入力（例: star.png）
  3. エフェクト JSON をエクスポートしてトリガーに読み込む

▶ 注意事項

  • アプリを起動したまま OBS を使用してください
  • Client Secret は不要です（入力しないでください）
  • ポートを変えた場合は Twitch コンソールのリダイレクト URI も更新が必要です

4. OBS への追加方法

  1. OBS の「ソース」パネルで「＋」をクリック →「ブラウザ」を選択
  2. URL に http://localhost:39080/display を入力
  3. 幅と高さを配信画面と同じ解像度にする（例: 1920 × 1080）
  4. 背景は自動で透過されるので、そのまま他のソースに重ねて OK

⚠ 画像一覧は設定保存時に記録されます。画像を追加・削除した場合は保存を押し直してください。
